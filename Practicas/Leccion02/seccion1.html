Para crear el contenido de una página web es necesario que todo el contenido de la misma este en una misma carpeta y una de sus paginas se llame 'index.html'.
La extensión .html permite que se pueda leer desde los navegadores y son los archivos que llevan todo el contenido, llamando todos los recursos necesarios en la visulización del contenido.

Estructura Básica de un Documento HTML

Todo documento de HTML tiene esta estructura básica:

  • <!DOCTYPE html>: Es la primera etiqueta que se debe escribrir sin la necesidad de cerrarla; esta indica que el documento que se va leer es de tipo html5.
  • <html>: Dentro esta etiqueta es donde se escribira todo el código de la página.
  • <head>: Es la etiqueta de cabecero, la información que contiene otorga privilegios a toda la página; como el titulo, links a hojas de estilo, links de imagenes, juego de caracteres del idioma que se vaya a usar y muchos más.
  • <body>: Es la etiqueta que contiene lo que se visualizara en el navegador web
Otras etiquetas usadas generalmente para diseño son:

Etiqueta Descripción
<p></p> Se utiliza para mostrar un párrafo; automatícamente se le aplican de espaciados antes y despúes del texto y se comporta como un objeto dentro de la página.
<h1...h7></h1...h7> Se utiliza para mostrar un titulo, los estilos de titulo van desde 1 hasta el 7; automatícamente se le aplican de espaciados antes y despúes del texto y se comporta como un objeto dentro de la página.
<meta charset="UTF-8"> Las 'meta etiquetas' o 'meta tags' no se necesita cerrarlas. Estas contienen metadatos (datos sobre datos) que describen el contenido del sitio web con precisión transmitiendo información estructurada. Los metadatos pueden variar dependiendo de la descripción de los atributos deseados, siempre van en el <head> del código, no estan a la vista del sitio y en HTML5 hay solo 5 atributos disponibles para la etiqueta que son: charset, http-equiv, nombre, esquema y contenido.
Por ejemplo, el atributo charset define que tipo de caracteres se utilizarán en la web, para que el navegador los detecte y visualice correctamente. El valor UTF-8 es el estandar internacional para los caracteres especiales del idioma español.
Observación: Todas las etiquetas tienen un valor semantico que les permite a los navegadores leer e interpretar eficientemente el código; hay que mencionar que algunas etiquetas se usan especialmente solo para valor semantico, de estas se hablara más adelante.
