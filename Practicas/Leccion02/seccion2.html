LECCION 2: Referencia de Caracteres

La etiqueta 'code' Se puede usar para códido de cierto lenguaje de programación.

Para asegurar el uso correcto de algún caracter se puede usar la referencia estandarizada por la organización W3 - 'https://html.spec.whatwg.org/multipage/named-characters.html'

Simbolo de 'check' ✓ ≡

Ingresando de manera hexagesimal &#x + valor hexagesimal

Emjemplo: con la combinación (&#x + 02713 + ;) tenemos ✓

Tambien se puede con valor decimal si el caracter cuenta con dicho valor;
la sintaxis: &# + valor decimal + ;

Otros simbolos:

Menor que <

Mayor que >

Arroba @

Vocales con tilde; Á, á, É, é, Í, í, Ó, ó, Ú, ú
