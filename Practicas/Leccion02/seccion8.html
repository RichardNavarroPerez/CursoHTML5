8.1. Formularios (Formas) en HTML

Las etiquetas Input son cajas que permiten capturar información de parte del usuario, y con el atributo 'Type' se especifica el tipo de elemento que se va a manejar; por buenas practicas se le debe asignar un 'id' a cada input que se cree.

Con el atributo 'value' asigna la expresiónque se muestra en el boton de envio, si esta etiqueta no se toma en cuenta el valor por defecto es 'Enviar'. Sin embargo, el atribito 'value' tambien se le puede asignar un valor que aparecera por decto en un 'input'.

Todo 'input' debe estar acompañado por un 'label' porque le dice al usuario que dato se espera capturar.
